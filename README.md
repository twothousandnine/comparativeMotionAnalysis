# Comparative Motion Analysis

For identification concern, data has been anonymized and removed of any semblance of the original subject. Only non-identifying data points were left in.
Enrivonment is provided via poetry. `pyproject.toml` lists all dependencies for reproduction of environment.
`compute_` scripts showcase the processing for all endpoint variables.
`fig_` and `table_` scripts correspond to the the figures and tables in the paper. Run these scripts with the poetry environment to reproduce the figures of the paper.
